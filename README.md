# JS DOM Practice

A collection of small exercises for practicing DOM manipulation with vanilla JavaScript. No frameworks — just HTML, CSS, and JS, focused on understanding how the browser DOM works under the hood.

## 📌 About

This repo is a personal playground for learning and practicing core DOM concepts, including:

- Selecting elements (`querySelector`, `getElementById`, etc.)
- Creating, updating, and removing elements
- Event handling (click, input, submit, keyboard events, etc.)
- Event delegation and bubbling
- Working with forms and validating input
- Manipulating classes, styles, and attributes
- Traversing the DOM tree (parent/child/sibling nodes)
- Building small interactive UI components (to-do lists, modals, tabs, sliders, etc.)

Each folder contains its own `index.html`, `style.css`, and `script.js` so exercises can be opened and run independently.

## 🚀 Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/your-username/js-dom-practice.git
   ```
2. Navigate into any exercise folder
   ```bash
   cd js-dom-practice/03-dom-manipulation
   ```
3. Open `index.html` in your browser (or use a tool like [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) for auto-reload)

No build tools, no dependencies — just open and go.
